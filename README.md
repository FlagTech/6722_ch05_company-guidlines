# Company Guidelines

## 📋 專案說明

本儲存庫為 **5-1 節遠端規則（Remote Rule）的教學範例**，示範如何使用 Cursor 的 Remote Rule 功能，將開發規範集中管理在 Git Repository 中，實現團隊協作時的規則統一。

## 🎯 功能特色

- **遠端規則管理**：將 Cursor 規則檔案存放在 Git Repository，而非本地端
- **團隊協作**：多人開發時能夠共享相同的 AI 輔助規則與編碼標準
- **版本控制**：透過 Git 追蹤規則的變更歷史，方便回溯與管理
- **即時同步**：團隊成員連接此 Repository 後，即可使用統一的開發規範
