# Company Guidelines

## 📋 專案說明

本儲存庫用於存放團隊的 Cursor 遠端規則（Remote Rules），配合 Cursor 的 Remote Rule 功能，讓團隊能夠將開發規範集中管理在 Git Repository 中。

## 🎯 用途

- **集中管理**：將 Cursor 的規則檔案統一存放在 Git Repository
- **統一規範**：確保團隊成員使用相同的開發規範和編碼標準
- **版本控制**：透過 Git 追蹤規則的變更歷史
- **多人協作**：團隊成員可以共同維護和更新開發規範

## 📁 目錄結構

```
.cursor/

```
